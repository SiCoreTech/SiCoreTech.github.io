NAME

initialize_file_system - initialize NTFS file system information related to the VBR and MFT

SYNOPSIS

#include <initialize.h> /* See NOTES */

int initialize_file_system(struct pte * p);

Compile and link with -DSATA_MEDIATOR when cross-compiling for SATA Mediator hardware.

DESCRIPTION

initialize_file_system() calls initialize_volume_info() from volume_info.h and initialize_mft_runlist() from mft_runlist.h in order to initialize important volume info macros and a global for the Master File table runlist allowing the user access to other routines for traversing and manipulating an NTFS partition. **Please see the man pages on initialize_volume_info() and intialize_mft_runlist() for further information.

RETURN VALUE

On success initialize_file_system() returns 0 and -1 on failure.

NOTES

All headers for the SATA NTFS project can be found in the include directory in the project repo. Compile with -I./include/.
