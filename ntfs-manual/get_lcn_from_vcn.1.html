NAME

get_lcn_from_vcn - convert Virtual Cluster Number in runlist to Logical Cluster Number

SYNOPSIS

#include <data_runs.h> /* See NOTES */

long long get_lcn_from_vcn(runlist r, unsigned long long vcn);

Compile and link with -DSATA_MEDIATOR when cross-compiling for SATA Mediator hardware. Depends upon the initialization of Volume Info macros and must be preceded by a call to initialize_volume_info().

DESCRIPTION

get_lcn_from_vcn() takes a runlist and Virtual Cluster Number as arguments and returns the Logical Cluster Number in relation to the start of the NTFS partition.

RETURN VALUE

returns a long long representing the Logical Cluster Number corresponding to the Virtual Cluster Number (unsigned long long vcn) argument. If the vcn is part of a sparse run or is invalid (i.e, it is negative or greater than the total cluster length of the runlist) then a -1 is returned;

NOTES

All headers for the SATA NTFS project can be found in the include directory in the project repo. Compile with -I./include/.
