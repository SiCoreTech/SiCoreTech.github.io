NAME

initialize_mft_runlist - read the non-resident runlist representative of the cluster locations of the entire Master File Table

SYNOPSIS

#include <mft_runlist.h> /* See NOTES */

int initialize_mft_runlist(void);

Compile and link with -DSATA_MEDIATOR when cross-compiling for SATA Mediator hardware.

DESCRIPTION

initialize_mft_runlist() uses the macro MFT_LBA from volume_info.h to retrieve the file entry $MFT. It then retrieves the data attribute (type id 0x80) from $MFT. This attribute is always in non-resident in $MFT and represent the contents of the entire Master File Table; initialize_mft_runlist() parses and stores the runlist of the data attribute in a global runlist MFT_RUNLIST for future use. For futher information on runlist please see get_runlist(). Dependent upon preceding call to initialize_volume_info() from volume_info.h

RETURN VALUE

On success initialize_mft_runlist() returns 0, on failure -1 is returned and errno is set appropriately:

EBADF MFT_LBA or Data Attribute are invalid.

NOTES

All headers for the SATA NTFS project can be found in the include directory in the project repo. Compile with -I./include/.
